Bài tập về nhà

Hoàn thành 5 mục dưới đây sau buổi workshop. Copy prompt, chạy trên IDE (Cursor / Claude Code / OpenCode), chụp màn hình kết quả và trả lời câu hỏi cuối.

  1. 1

    Cài đặt Minipower

    Mở folder dự án trống trên IDE, paste prompt bên dưới và chờ AI clone repo cùng chạy setup. Không chuyển sang bài tiếp theo khi chưa xong bước này.

    Prompt
    Pull code https://github.com/hoangnh2412/ai-skills và cài đặt minipower
  2. 2

    Chụp kết quả AI thông báo thành công

    Sau khi AI cài xong, chụp màn hình thông báo thành công từ AI (ví dụ: đã clone repo, đã cài skill, setup hoàn tất…).

    Click hoặc kéo thả ảnh chụp màn hình vào đây

  3. 3

    Chụp kết quả hook & rules sau khi kiểm tra

    Dùng prompt kiểm tra bên dưới, xác nhận đã cài đủ skill, rules và hook guardrails. Chụp màn hình kết quả kiểm tra (hoặc folder cấu hình nếu tự mở thủ công).

    Prompt kiểm tra
    Làm thế nào để biết đã cài: Skill minipower, rules, hook guardrails
    • Skill minipower
    • rules
    • hook guardrails
    Kiểm tra thủ công — đừng tin AI 100%. Không cài đủ có thể tốn rất nhiều token.

    Click hoặc kéo thả ảnh chụp màn hình vào đây

  4. 4

    Chụp kết quả prompt “/minipower bước tiếp theo”

    Sau khi cài xong, chạy prompt dưới đây và chụp màn hình phản hồi của AI.

    Prompt
    /minipower bước tiếp theo cần làm gì? Ko cần cả quá trình, dẫn dắt tôi làm từng bước

    Click hoặc kéo thả ảnh chụp màn hình vào đây

  5. 5

    Trả lời câu hỏi

    Bạn mong muốn Minipower giải quyết vấn đề gì của bạn? Viết ngắn gọn, cụ thể — nêu bối cảnh dự án hoặc nỗi đau thực tế bạn đang gặp.

Nộp bài: Gửi link hoặc file ảnh chụp màn hình + câu trả lời bài 5 cho giảng viên qua kênh đã thông báo trong buổi học. Trang này lưu tạm ảnh và câu trả lời trên trình duyệt của bạn (localStorage).
